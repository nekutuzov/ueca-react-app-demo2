## 🚀 What is UECA-React?

UECA-React extends React with explicit component IDs, automatic two-way property bindings, controlled lifecycle hooks, reactive state, automatic on-change events for model properties, and a built-in message bus. These features provide the clarity and control large, complex web applications require.

---

## ⚡ Why It Excels at Scale

- **Predictable architecture**: Every component has a unique ID and clear mental model. No hidden dependencies — teams read and maintain the code with ease.
- **Zero-boilerplate data flow**: Automatic property bindings sync state to UI instantly. Changes propagate reliably without manual setters.
- **Controlled lifecycle**: Precise hooks — `constr`, `init`, `deinit`, `mount`, `unmount` — give full control over creation, activation, cleanup, and DOM lifecycle.
- **Automatic events**: For every property, `onChange<Prop>` and `onChanging<Prop>` events are generated automatically, enabling clean reactive logic.
- **Stable IDs for testing**: Component IDs produce consistent HTML IDs with tracing into source code, perfect for automated UI testing with tools like UI Vision and troubleshooting with rendering.
- **Decoupled communication**: The message bus allows components to interact without tight coupling — ideal for dashboards, forms, and multi-module systems.

**The result**: faster development, fewer bugs, and effortless scaling even in large codebases.

## 🎯 Explore the Demo

This application functions as an interactive storybook, showcasing each UI component with live examples and configurable properties. Navigate through the menu to explore:

- **Layout components** (Block, Row, Col, Card) with spacing, alignment, and styling options
- **Input components** (Button, TextField, Select, RadioGroup, Checkbox) with variants, sizes, and validation
- **Navigation components** (NavLink, NavItem) with different modes and states
- **Popup components** (Dialog, Drawer, Toast) with various configurations

- **Property Editor**: Interactively modify component properties in real-time
